Metric Spaces
Definitions · Properties · Key Theorems
Def Metric Space
Definition. A metric space is a pair (X, d) where X is a set and d: X×X → ℝ satisfies:
1. d(x,y) ≥ 0; d(x,y)=0 ⟺ x=y (non-neg) 2. d(x,y) = d(y,x) (symmetry) 3. d(x,z) ≤ d(x,y)+d(y,z) (triangle ineq)

Common metrics: Euclidean d₂, taxicab d₁, supremum d∞, discrete metric.

Def Open & Closed Sets
Open ball. B(x,r) = {y ∈ X : d(x,y) < r}
Open set U: ∀x ∈ U, ∃r>0 s.t. B(x,r) ⊆ U
Closed set: complement is open ⟺ contains all its limit points
  • ∅ and X are both open and closed
  • Arbitrary unions of open sets are open
  • Finite intersections of open sets are open
Def Sequences & Limits
xₙ → x iff ∀ε>0, ∃N s.t. n ≥ N ⟹ d(xₙ, x) < ε
  • Limits are unique in a metric space
  • Convergent sequences are Cauchy
  • Cauchy: d(xₘ,xₙ)→0 as m,n→∞

⚠ Cauchy does not imply convergent unless the space is complete.

Thm Completeness
Complete metric space: Every Cauchy sequence converges in X.
  • ℝⁿ with Euclidean metric is complete
  • ℚ with |·| is not complete
  • Closed subsets of complete spaces are complete
Banach Fixed Point Theorem: If T: X→X is a contraction (∃k<1: d(Tx,Ty)≤k·d(x,y)) on a complete space, then T has a unique fixed point.
Def Compactness
Compact: every open cover has a finite subcover.
Heine-Borel (ℝⁿ): A ⊆ ℝⁿ is compact ⟺ closed and bounded.
  • Compact ⟹ complete and totally bounded
  • Continuous image of compact set is compact
  • Compact set ⟹ every sequence has convergent subsequence (sequentially compact)
Key Continuity
Continuous at x: ∀ε>0, ∃δ>0 s.t. d(x,y)<δ ⟹ d(f(x),f(y))<ε
Equivalent characterisations: • xₙ→x ⟹ f(xₙ)→f(x) • f⁻¹(open) is open • f⁻¹(closed) is closed
Uniform continuity: δ depends only on ε, not on x. Continuous on compact ⟹ uniformly continuous.
Thm Connectedness
Connected: Cannot write X = U ∪ V with U,V open, disjoint, non-empty.
  • Intervals in ℝ are connected
  • Continuous image of connected set is connected
  • Path-connected ⟹ connected (converse fails)
Intermediate Value Theorem: f continuous, X connected ⟹ f(X) is an interval.
Def Topology Vocab
  • Interior A°: largest open set ⊆ A
  • Closure Ā: smallest closed set ⊇ A
  • Boundary ∂A = Ā \ A°
  • Dense: Ā = X (e.g. ℚ dense in ℝ)
  • Separable: has countable dense subset
  • Nowhere dense: int(Ā) = ∅
Chapter 1: Introduction to Metric Spaces
Distances · Examples · ℓᵖ Spaces · Function Spaces · Isometries
Def 1.1 Metric (Distance Function)
Definition. A metric on a set X is d : X×X → ℝ≥0 satisfying:
(M0) d(x,y) ≥ 0 (non-negativity) (M1) d(x,y) = 0 ⟺ x = y (identity) (M2) d(x,y) = d(y,x) (symmetry) (M3) d(x,z) ≤ d(x,y)+d(y,z) (triangle ineq)

⚠ Remark 1.2: Don't forget (M0)! Always check non-negativity separately from (M1).

Ex 1.8–1.11 Standard Metrics on ℝⁿ
Euclidean (d₂): d₂(x,y) = √(Σ|xᵢ−yᵢ|²) Manhattan / Taxicab (d₁): d₁(x,y) = Σ|xᵢ−yᵢ| Supremum / Chebyshev (d∞): d∞(x,y) = max₁≤ᵢ≤ₙ |xᵢ−yᵢ| General Lᵖ (Def 1.21): dₚ(x,y) = (Σ|xᵢ−yᵢ|ᵖ)^(1/p)

For n=1: d₁ = d₂ = |x−y|. All dₚ are valid metrics on ℝⁿ via Minkowski's inequality.

Prop 1.12 Cauchy–Schwarz Inequality
Statement. For any a₁,…,aₙ, b₁,…,bₙ ∈ ℝ:
(Σ aᵢbᵢ)² ≤ (Σ aᵢ²) · (Σ bᵢ²)
Proof idea. Define F(t) = Σ(aᵢt + bᵢ)² ≥ 0. This is quadratic in t with discriminant H²−4AB ≤ 0, which gives the inequality.

Used to prove the triangle inequality for d₂ (Euclidean distance) via the identity d₂(x,z)² = (a₁+b₁)²+(a₂+b₂)² and bounding cross terms.

Thm 1.39 Minkowski Inequality
Statement. For sequences (Aₙ), (Bₙ) and p ≥ 1:
(Σ|Aₙ+Bₙ|ᵖ)^(1/p) ≤ (Σ|Aₙ|ᵖ)^(1/p) + (Σ|Bₙ|ᵖ)^(1/p)
Proof idea. Normalise: Âₙ = Aₙ/A, B̂ₙ = Bₙ/B. Use convexity of F(t)=tᵖ with α=A/(A+B), β=B/(A+B), then sum and take p-th roots.
  • p=2 case recovers Cauchy–Schwarz (Remark 1.20)
  • Holds for all real p ∈ [1,∞), not just integers
  • Used to prove triangle inequality for all dₚ (Thm 1.22)
Ex 1.24 Discrete Metric
Definition. For any set X:
ddiscr(x,y) = 0 if x=y, 1 if x≠y
Triangle inequality. If x≠z, then for any y, at least one of x≠y or y≠z must hold, so the RHS is ≥ 1.

Ex 1.58: Any bijection between sets X and Y is an isometry for (X, ddiscr) and (Y, ddiscr). So set-theoretic size is all that matters.

Def 1.32 ℓᵖ Sequence Spaces
For 1 ≤ p < ∞: ℓᵖ = { (aₙ) : Σ|aₙ|ᵖ < ∞ } ‖x‖ₚ = (Σ|xₙ|ᵖ)^(1/p) For p = ∞: ℓ∞ = { (aₙ) : sup|aₙ| < ∞ } ‖x‖∞ = sup|xₙ|
Thm 1.34. For all p ≥ 1 (incl. p=∞), dₚ is a distance on ℓᵖ.
Prop 1.35 (Inclusions). p ≤ q ⟹ ℓᵖ ⊆ ℓq.

Example: (1,−1,1,−1,…) ∈ ℓ∞ but ∉ ℓᵖ for finite p. The harmonic sequence (1/n) ∈ ℓ² but ∉ ℓ¹.

§1.11 Function Spaces C([0,1])
Definition 1.41. C([0,1]) = { f : [0,1] → ℝ | f continuous }.
L∞ metric (Def 1.42): dL∞(f,g) = max_{x∈[0,1]} |f(x)−g(x)| L¹ metric (Def 1.42): dL1(f,g) = ∫₀¹ |f(x)−g(x)| dx
Example 1.47. For f(x)=x, g(x)=x²:
dL∞(f,g) = 1/4 (max at x=1/2) dL1(f,g) = 1/6 (∫₀¹(x−x²)dx)
§1.12 Product Metrics on X×Y
Setup. (X,dX) and (Y,dY) metric spaces. Three natural metrics on X×Y:
D₁((x₁,y₁),(x₂,y₂)) = dX(x₁,x₂) + dY(y₁,y₂) D₂((x₁,y₁),(x₂,y₂)) = √(dX²+dY²) D∞((x₁,y₁),(x₂,y₂)) = max{dX(x₁,x₂), dY(y₁,y₂)}
Thm 1.49. D₂ is a distance. Proof uses Cauchy–Schwarz to bound the cross term in the triangle inequality.
Def 1.51 Isometry
Definition. φ: (X,dX) → (Y,dY) is an isometry if:
  • dY(φ(x₁), φ(x₂)) = dX(x₁, x₂) for all x₁,x₂ ∈ X
  • φ is surjective
Prop 1.52. Every isometry is automatically injective (hence bijective). Proof: φ(x₁)=φ(x₂) ⟹ dY=0 ⟹ dX=0 ⟹ x₁=x₂.
Key examples: Translations, rotations, reflections are isometries of (ℝⁿ, d₂).
Ex 1.54–1.63 Isometry Pitfalls
  • ([0,1], d₁) ≅ ([2,3], d₁) via φ(x) = x+2 ✓
  • ([0,2], d₁) ≇ ([0,1], d₁): no two points in [0,1] have distance 2
  • Same set, different metric: (A,d₂) ≇ (B,d₂) for square vs disk (max dist differs), but (A,ddiscr) ≅ (B,ddiscr) via any bijection
  • (ℝ, ddiscr) ≅ (ℝ², ddiscr) since any bijection works
  • (ℝ, d₁) ≇ (ℝ², d₂): ℝ² has equilateral triangles of side 1; ℝ does not
  • Shift R(a₀,a₁,…) = (0,a₀,a₁,…) is an isometry onto its image in ℓᵖ
Ex 1.25 / Def 1.5 Subspaces & Cartesian Products
Subspace metric (Ex 1.25). If (X,dX) is a metric space and Y ⊆ X, then dY = dX|Y×Y is a metric on Y. All axioms inherit from X.
Cartesian product (Def 1.5). X×Y = {(x,y) | x∈X, y∈Y}. If X,Y finite: #(X×Y) = #X·#Y.
Extension to n dimensions: ℝⁿ = ℝ×ℝ×···×ℝ (n times) x = (x₁, x₂, …, xₙ)

Remark 1.28: ℝⁿ embeds in ℓᵖ via (x₀,…,xₙ₋₁) ↦ (x₀,…,xₙ₋₁,0,0,…).

Key Results Quick Reference
  • d₁ on ℝ (Ex 1.8): d₁(x,y)=|x−y|. Triangle ineq. by |a+b|≤|a|+|b|.
  • dₚ ordering (Ex 1.23): p ≤ q ⟹ dₚ(x,y) ≥ dq(x,y) for fixed x,y∈ℝⁿ
  • dₚ → d∞ (Ex 1.23): lim_{p→∞} dₚ(x,y) = d∞(x,y)
  • Geometric seq (Ex 1.37): (xⁿ) ∈ ℓᵖ (p<∞) iff |x|<1; ∈ ℓ∞ iff |x|≤1
  • Lemma 1.46: If h∈C[0,1] and ∫₀¹h=0, then h≡0. Used to prove (M1) for dL1.
  • L∞ triangle ineq.: max attained at some x' by continuity; then apply pointwise triangle ineq.
Chapter 2: Continuity & Convergence
From your lecture notes · Definitions · Propositions · Key Examples
Def 2.4 Convergence in (X, d)
Sequence convergence. (xₙ) converges to ℓ ∈ X if:
∀ε>0, ∃N ∈ ℤ₊ such that ∀n > N, d(xₙ, ℓ) < ε Notation: lim xₙ = ℓ, or xₙ → ℓ, or xₙ →(d) ℓ

Note: strict inequality n > N — match your lecturer's notation in proofs.

Def 2.7 Open & Closed Balls
Open ball: B_R(p) = {x ∈ X | d(x,p) < R} Closed ball: B̄_R(p) = {x ∈ X | d(x,p) ≤ R}
  • d₂ in ℝ²: open disk √(x₁²+x₂²) < R
  • d∞ in ℝ²: open square max{|x₁|,|x₂|} < R
  • d₁ in ℝ²: open diamond |x₁|+|x₂| < R
  • Discrete metric: B_R(x₀) = {x₀} if R≤1, else X
⚠ Exam Trap ℓ∞ Closed Ball
Example 2.11. In (ℓ∞, d∞) with A = (0,0,…), R=1:
Open ball: x ∈ B₁(A) ⟺ sup|xₙ| < 1 Closed ball: x ∈ B̄₁(A) ⟺ sup|xₙ| ≤ 1 ⚠ "∀n, |xₙ|<1" does NOT imply sup|xₙ|<1 Counterex: xₙ = 1−1/n has each term <1 but sup = 1, so it's in B̄₁ not B₁
Def 2.12 Eventually Constant
Definition. (xₙ) is eventually constant if ∃ℓ, ∃N s.t. ∀n≥N, xₙ = ℓ.
Example 2.13. Under the discrete metric ddiscr, a sequence converges ⟺ it is eventually constant.
Def 2.14 (reformulation). xₙ → ℓ iff ∀ε>0, ∃N s.t. ∀n>N: xₙ ∈ Bε(ℓ)
Examples Convergence in C[0,1]
Ex 2.15 (converges in d∞). fₙ(x) = x/(n+1), f = 0:
d∞(fₙ,f) = max|x/(n+1)| = 1/(n+1) → 0 ✓
Ex 2.16 (diverges in d∞). fₙ(x) = xⁿ, f = 0:
fₙ(1)=1, f(1)=0 ⟹ d∞(fₙ,f) ≥ 1 for all n ⟹ fₙ ↛ f in d∞ (though fₙ→f pointwise)
§2.3 Sequences in ℓᵖ
Key idea. Elements of ℓᵖ are sequences, so a sequence in ℓᵖ is a "sequence of sequences." Write as matrix Aₖ = (Aₖ,ₙ)ₙ, row k.
Ex 2.17 — CONVERGES in ℓ∞: Aₖ,ₙ = (-1)ⁿ/(k+1) sup|Aₖ,ₙ| = 1/(k+1) → 0 ⟹ Aₖ → 0 Ex 2.18 — DIVERGES in ℓ∞: Aₖ,ₙ = (-1)ᵏ/(n+1) Rows alternate ±(1,½,⅓,…) ⟹ no limit (B₀ must be both >0 and <0 — contradiction)
Prop 2.19 Uniqueness of Limits
Statement. If xₙ → ℓ₁ and xₙ → ℓ₂ in (X,d), then ℓ₁ = ℓ₂.
Proof sketch. Set ε = d(ℓ₁,ℓ₂)/2. Use triangle inequality for n ≥ max{N₁,N₂}:
d(ℓ₁,ℓ₂) ≤ d(ℓ₁,xₙ)+d(xₙ,ℓ₂) < ε+ε = d(ℓ₁,ℓ₂) Contradiction ⟹ ℓ₁ = ℓ₂ □
Def 2.21 Continuity: (X,dₓ) → (Y,dᵧ)
f continuous at x₀ if:
∀ε>0, ∃δ>0: dₓ(x,x₀)<δ ⟹ dᵧ(f(x),f(x₀))<ε Equivalent (Prop 2.25): f(B^dₓ_δ(x₀)) ⊆ B^dᵧ_ε(f(x₀))
Limit (Def 2.22). lim f(x) = ℓ uses strict 0 < dₓ(x,x₀) < δ.

Continuous at x₀ ⟺ lim_{x→x₀} f(x) = f(x₀).

Prop 2.26 Composition is Continuous
Theorem. f: X→Y continuous at x₀, g: Y→Z continuous at f(x₀) ⟹ g∘f continuous at x₀.
Proof (ball language).
∃δ₂: g(B_{δ₂}(f(x₀))) ⊆ Bε(g(f(x₀)))
∃δ₁: f(B_{δ₁}(x₀)) ⊆ B_{δ₂}(f(x₀))
∴ (g∘f)(B_{δ₁}(x₀)) ⊆ Bε(g(f(x₀))) □

Corollary 2.27: If f and g are both continuous everywhere, so is g∘f.

Prop 2.28 Sequential Continuity
Theorem. f: X→Y is continuous at p ⟺ for every sequence xₙ → p in X, f(xₙ) → f(p) in Y.
  • To prove continuity: show all convergent sequences map correctly
  • To prove discontinuity: find ONE sequence xₙ→p with f(xₙ)↛f(p)
Example 2.30. Φ(f) = max f on (C[0,1], d₁) is NOT continuous: take fₙ = xⁿ →_{d₁} 0, but Φ(fₙ) = 1 ↛ 0 = Φ(0).

Exercise 2.31: Show Φ IS continuous w.r.t. d∞ — try it!

Chapter 3: Topology of Metric Spaces
Open & Closed Sets · Topology · Equivalent Metrics · Homeomorphisms
Def 3.1 Open & Closed Sets
Open: A ⊆ X is open if ∀p ∈ A, ∃ε>0: Bε(p) ⊆ A
Closed: S ⊆ X is closed if X \ S is open
  • (0,1) is open in ℝ; [0,1] is closed in ℝ
  • [0,1) is neither open nor closed in ℝ
  • ℝ and ∅ are both open AND closed in any metric space
  • [0,1) IS open in ([0,∞), d₁) — openness depends on the ambient space!
Prop 3.5 Properties of Open Sets
1. ∅ and X are open.
2. Arbitrary unions of open sets are open.
3. Finite intersections of open sets are open.
⚠ "Finite" matters! Example 3.7: Aₖ = (−1/k, 1/k) are open, but ∩Aₖ = {0} is NOT open.
Proof key (part 3): Take ε = min(ε₁,…,εₙ) > 0. Then Bε(p) ⊆ ∩Aᵢ. Minimum of finitely many positive numbers is positive — fails for infinite collections.
Cor 3.8 Properties of Closed Sets
1. ∅ and X are closed.
2. Arbitrary intersections of closed sets are closed.
3. Finite unions of closed sets are closed.
⚠ "Finite" matters! Example 3.9: Aε = [0,ε] is closed for each ε>0, but ∪_{ε∈(0,1)} Aε = [0,1) is NOT closed.

These follow from Prop 3.5 by De Morgan's laws: complement of union = intersection of complements.

Prop 3.10 Sequential Characterisation of Closed
Theorem. A ⊆ X is closed ⟺ for every sequence (xₙ) in A with xₙ → ℓ in X, we have ℓ ∈ A.
  • To prove A is NOT closed: find (xₙ) ⊆ A with xₙ → ℓ ∉ A
  • ℚ is not closed in ℝ: rationals approximate π, but π ∉ ℚ
  • ℓᵖ is not closed in ℓᵍ (p<q): truncations Aₖ ∈ ℓᵖ converge in dᵧ to B ∉ ℓᵖ
  • [−1,1) not closed: xₙ = 1−1/(n+1) → 1 ∉ [−1,1)
Def 3.12 Topology (Abstract)
Topology U on X: a collection of subsets (called open sets) satisfying:
(T1) ∅, X ∈ U (T2) Arbitrary unions of elements of U ∈ U (T3) Finite intersections of elements of U ∈ U
  • Discrete topology: U = P(X) — every subset is open (largest)
  • Trivial topology: U = {∅, X} (smallest)
  • Metric topology: open sets of (X,d) form a topology (Example 3.16)
  • On a 3-point set: exactly 29 distinct topologies exist
Thm 3.20 Continuity via Open Sets
Theorem. f: X → Y is continuous ⟺ for every open U ⊆ Y, f⁻¹(U) is open in X.
Remark 3.21. Equivalently: for every closed C ⊆ Y, f⁻¹(C) is closed in X.
Proof key (⟸): Given ε>0, Bε(f(x₀)) is open ⟹ f⁻¹(Bε(f(x₀))) is open ⟹ ∃δ: Bδ(x₀) ⊆ f⁻¹(Bε(f(x₀))).

This is the topological definition of continuity — works for any topological space, not just metric spaces.

Thm 3.19 Convergence via Open Sets
Theorem. xₙ → ℓ ⟺ for every open U containing ℓ, ∃N s.t. ∀n>N: xₙ ∈ U.
Example 3.23. To show xₙ = (n+1)/n, sin(1/n)) → (1,0) in (ℝ², d₂):
d₂(xₙ,(1,0)) = √(1/n² + sin²(1/n)) Use |sin(1/n)| ≤ 1/n: ≤ √(1/n² + 1/n²) = √2/n Take N = ⌊√2/ε⌋ + 1 ⟹ √2/n < ε ✓
Def 3.24 Equivalent Metrics
d ~ d' if (X,d) and (X,d') have the same open sets.
Corollary 3.29 (Practical test). If ∃C,C'>0 s.t. d ≤ C·d' AND d' ≤ C'·d, then d ~ d'.
  • All dₚ metrics on ℝⁿ are equivalent (Cor 3.30): d∞ ≤ dₚ ≤ d₁ ≤ n·d∞
  • d₁ ~ d∞ on ℝⁿ, but NOT on ℓ¹ (Example 3.34)
  • d₁ ≤ d∞ on C[0,1] (Prop 3.31), but d₁ ≁ d∞ (xⁿ → 0 in d₁ but not d∞)
  • ddiscr ≁ d₁ on ℝ: [0,1] is open in ddiscr but not d₁
Prop 3.28 One-sided Inequality Trick
Proposition. If d(x,y) ≤ C·d'(x,y) for all x,y, then: U open in (X,d) ⟹ U open in (X,d').
Proof idea: U open in d ⟹ ∃R: B^d_R(p) ⊆ U Set r = R/C. If d'(x,p) < r then d(x,p) ≤ C·d'(x,p) < C·r = R So B^{d'}_r(p) ⊆ B^d_R(p) ⊆ U □

Two such inequalities (both directions) ⟹ d ~ d' by Corollary 3.29.

Def 3.36 Homeomorphism
f: X → Y is a homeomorphism if f is bijective and both f and f⁻¹ are continuous.
  • Every isometry is a homeomorphism
  • [0,1] ~ [0,2] via f(x)=2x (homeomorphic, not isometric)
  • (−π/2, π/2) ~ ℝ via tan (homeomorphic, not isometric)
  • [0,1] and ℝ are NOT homeomorphic
  • "O" and "D" are homeomorphic; "I" and "P" are not
Example 3.39. Id: (X,d)→(X,d') is a homeomorphism ⟺ d ~ d'.
Chapter 4: Completeness & Compactness
Cauchy Sequences · CMT · Bolzano–Weierstrass · Compactness
Def 4.1 / 4.4 Cauchy & Completeness
Cauchy sequence: ∀ε>0, ∃N s.t. ∀m,n ≥ N: d(xₙ,xₘ) < ε
Prop 4.2: Convergent ⟹ Cauchy. (Converse fails in general!)
Complete: every Cauchy sequence converges.
  • (ℚ, d₁) not complete: decimal approx of π is Cauchy but π ∉ ℚ
  • ((0,1), d₁) not complete: xₙ = 1−1/(n+1) → 1 ∉ (0,1)
  • (ℝⁿ, dₚ), (C[0,1], d∞), (ℓᵖ, dₚ) all complete
  • (X, ddiscr) complete: Cauchy ⟹ eventually constant ⟹ converges
Thm 4.6 Completeness & Closed Sets
1. (A, dₐ) complete ⟹ A is closed in (X, d)
2. (X, d) complete + A closed ⟹ (A, dₐ) complete
Proof key (1): Any convergent sequence in A is Cauchy ⟹ converges in A (completeness) ⟹ limit in A ⟹ A closed by Prop 3.10.
Proof key (2): Cauchy in A ⟹ Cauchy in X ⟹ converges in X ⟹ limit in A (since A closed).

⚠ Example 4.7: completeness is NOT preserved by homeomorphisms. (−π/2, π/2) ≅ ℝ but only ℝ is complete.

⚠ Ex 4.8 (C[0,1], d₁) is NOT Complete
Key example. Define fₙ as the piecewise-linear function ramp from 0 to 1 on [n/2(n+2), 1/2], constant otherwise.
  • fₙ is Cauchy in d₁: d₁(fₙ,fₘ) ≤ 1/(n+2) → 0
  • Pointwise limit g is the step function (0 on [0,½), 1 on (½,1])
  • g ∉ C[0,1] (discontinuous at x=½)
  • ∴ (C[0,1], d₁) is incomplete

Contrast: (C[0,1], d∞) IS complete — the metric matters!

Thm 4.14 / Cor 4.17 ℝⁿ is Complete
Proof strategy for ℝ (Thm 4.14): Let A = limsup aₙ, A = liminf aₙ. Show A = A using Cauchy property (terms eventually within ε of each other ⟹ sup−inf ≤ ε ⟹ A−A = 0).
Prop 4.15: xₙ → ℓ in (ℝᴺ, dₚ) ⟺ each coordinate xⱼ,ₙ → ℓⱼ in ℝ.
Cor 4.17: (ℝᴺ, dₚ) is complete for all N ≥ 1, p ≥ 1. Proof: reduce to coordinates, each is Cauchy in ℝ, converges by Thm 4.14.
Def 4.10 limsup & liminf
limsup Aₙ = lim_{m→∞} sup_{n≥m} Aₙ liminf Aₙ = lim_{m→∞} inf_{n≥m} Aₙ
Lemma 4.13: Aₙ → ℓ ⟺ limsup Aₙ = liminf Aₙ = ℓ
Example 4.11: Aₙ = (-1)ⁿ(n+2)/(n+1) limsup = 1, liminf = −1 ⟹ does not converge

Always: limsup ≥ liminf. Equal iff sequence converges.

Thm 4.22 Contraction Mapping Theorem
Contraction (Def 4.20): ∃L ∈ [0,1): d(f(x),f(y)) ≤ L·d(x,y) for all x,y
CMT: If (X,d) is complete and f is a contraction, then f has a unique fixed point.
Proof sketch: Uniqueness: d(p,q) ≤ L·d(p,q) ⟹ d(p,q)=0 Existence: iterate x₀, f(x₀), f²(x₀), … d(xₙ,xₘ) ≤ Lᵐ·d(x₁,x₀)/(1−L) → 0 Cauchy ⟹ converges to ρ; f(ρ)=ρ by continuity

⚠ d(f(x),f(y)) < d(x,y) (strict but not uniform) is NOT enough — see Example 4.24: f(x)=x+1/x has no fixed point on (1,∞).

Prop 4.21 Contraction via Derivative
Sufficient condition: If f: [a,b]→[a,b] is differentiable and |f'(x)| ≤ L < 1 for all x, then f is a contraction.
Proof: MVT ⟹ |f(x₂)−f(x₁)| = |f'(ξ)|·|x₂−x₁| ≤ L|x₂−x₁|.
Example 4.26: f(x) = x/2 + 3/4 on [0, 3/2] f'(x) = 1/2 ≤ 3/4 < 1 ⟹ contraction Unique fixed point: x = 1 (f(1) = 1 ✓)
Def 4.30 Compactness (Sequential)
Compact: every sequence has a convergent subsequence (with limit in X).
  • Finite sets are compact
  • Compact ⟹ closed (Prop 4.36) and bounded (Prop 4.32)
  • Compact ⟹ complete (Cor 4.43)
  • Complete ↛ compact: ℝ is complete but not compact
  • Continuous image of compact set is compact (Thm 4.33)
  • Compactness IS preserved by homeomorphism (Cor 4.34)

Contrast with completeness: completeness is NOT preserved by homeomorphism (Ex 4.7).

Thm 4.37/4.39 Bolzano–Weierstrass
In ℝ and ℝⁿ: X is compact ⟺ X is closed AND bounded.
Proof (⟸) for ℝⁿ: Extract subsequence with convergent 1st coordinates, then sub-subsequence with convergent 2nd coordinates, … After n steps all coordinates converge ⟹ sequence converges ⟹ limit in X (closed).
  • Closed ball in (ℝⁿ, dₚ) is compact
  • Open ball in (ℝⁿ, dₚ) is NOT compact
  • B₁(0) in (ℓᵖ, dₚ) is NOT compact: standard basis vectors eₙ have d(eᵢ,eⱼ)=√2 for i≠j — no convergent subsequence
Cor 4.35 Extreme Value Theorem
Theorem. If (X,d) is compact and f: X→ℝ is continuous, then f attains its maximum and minimum on X.
Proof: f(X) is compact (Thm 4.33) ⟹ f(X) closed and bounded in ℝ ⟹ sup and inf are in f(X) (Prop 4.32).
Prop 4.42 (bonus): If (xₙ) is Cauchy and some subsequence (xₙₖ) → ℓ, then the whole sequence (xₙ) → ℓ.

This is the key lemma linking compactness and completeness in the proof of Cor 4.43.

Chapter 5: Spaces of Continuous Functions
Uniform Convergence · Power Series · Differential Equations (Peano–Picard)
Def 5.1 Pointwise vs Uniform Convergence
Pointwise (weaker): ∀x ∈ X, lim fₙ(x) = f(x) Uniform (stronger): ∀ε>0, ∃N: ∀n≥N, sup_x |fₙ(x)−f(x)| < ε
Remark 5.2. Uniform ⟹ pointwise. Converse FAILS.
Example 5.3. fₙ(x) = xⁿ on [0,1] converges pointwise to f(x)=0 (x<1), f(1)=1, but sup|fₙ−f|=1 for all n, so no uniform convergence.

Remark 5.5: fₙ → f uniformly ⟺ fₙ → f in the dL∞-metric on B(X).

Thm 5.7 (B(X), dL∞) is Complete
Def 5.4. B(X) = {f : X → ℝ | f bounded}, with dL∞(f,g) = sup|f(x)−g(x)|.
Proof sketch (3 steps).
  • Step 1. (fₙ(x)) is Cauchy in ℝ for each x, so define f(x) = lim fₙ(x) pointwise.
  • Step 2. f is bounded: use ε=½ to get |f(x)−fM(x)| ≤ ½ for large M, so |f(x)| ≤ ½ + ‖fM‖.
  • Step 3. fₙ → f uniformly: Cauchy gives |fₙ(x)−fm(x)| < ε; let m→∞ to get |fₙ(x)−f(x)| ≤ ε.

⚠ Ex 5.6: Boundedness is essential — f(x)=1/x on (0,1] has dL∞(f,0)=+∞.

Thm 5.10 + Cor 5.11–5.13 Uniform Limit is Continuous
Thm 5.10. If fₙ are continuous and fₙ → f uniformly, then f is continuous.
Proof (ε/3 trick). Fix x₀, ε>0. Choose N so |fN(x)−f(x)| < ε/3 for all x. Use continuity of fN to get δ with |fN(x)−fN(x₀)| < ε/3. Then by triangle inequality |f(x)−f(x₀)| < ε.
  • Cor 5.11: BC(X) is closed in (B(X), dL∞)
  • Cor 5.12: (BC(X), dL∞) is complete (closed subset of complete space)
  • Cor 5.13: If X is compact, C(X)=BC(X), so (C(X), dL∞) is complete
Thm 5.15 Uniform Conv. ↔ Interchange ∫ and lim
Statement. If fₙ → f uniformly on [a,b], then:
lim_{n→∞} ∫_{x₁}^{x₂} fₙ(x)dx = ∫_{x₁}^{x₂} f(x)dx
Proof idea. Show I(f) = ∫f is continuous (dL∞ → d on ℝ) with δ = ε/|x₂−x₁|. Then apply continuity of I to the uniform limit.
⚠ Ex 5.16 (pointwise NOT enough). fₙ = n·𝟙(0,1/n): fₙ→0 pointwise but ∫fₙ=1 ↛ 0.
Thm 5.18 Uniform Conv. + Derivatives
Statement. If fₙ → f uniformly and f'ₙ continuous and f'ₙ → g uniformly, then f is differentiable and f' = g.
f(x) = lim fₙ(x) = lim[fₙ(a) + ∫_a^x f'ₙ] = f(a) + ∫_a^x g(t)dt ∴ f'(x) = g(x) (by FTC)
FTC (Thm 5.17). If φ continuous: F(x)=∫_a^x φ ⟹ F'=φ. If G'=φ ⟹ ∫_a^b φ = G(b)−G(a).
§5.2 Power Series & Radius of Convergence
Def 5.19. For Σaₙxⁿ, the radius of convergence is:
R = 1 / limsup_{n→∞} |aₙ|^(1/n) (R=∞ if limsup=0; R=0 if limsup=∞)
Thm 5.21. For every δ ∈ (0,R), the partial sums SN form a Cauchy sequence in (C([−δ,δ]), dL∞).
Cor 5.22. The power series defines a continuous function S : (−R,R) → ℝ.

Remark 5.28: For |x| > R, the series diverges (limsup of |aₙxⁿ|^(1/n) > 1).

Cor 5.24 Differentiating Power Series
Statement. If S(x) = Σaₙxⁿ has radius R > 0, then:
1. S'(x) = Σ(n+1)aₙ₊₁xⁿ (radius R) 2. ∫S(x)dx = Σ aₙ/(n+1) · xⁿ⁺¹ + C (radius R)
Example 5.25. Σnxⁿ⁻¹ = d/dx [Σxⁿ] = d/dx [x/(1−x)] = 1/(1−x)², for |x|<1.

Key: limsup |nan|^(1/n) = limsup |aₙ|^(1/n) since lim n^(1/n) = 1, so radius is unchanged by differentiation.

Prop 5.26 Enhanced Root Test
Statement. For (aₙ) ⊂ ℝ≥0:
  • limsup √ⁿaₙ < 1 ⟹ Σaₙ converges
  • limsup √ⁿaₙ > 1 ⟹ Σaₙ diverges
  • limsup √ⁿaₙ = 1 ⟹ inconclusive (Σ1/n diverges, Σ1/n² converges)
Proof idea (convergence case). If limsup < L < 1, then aₙ ≤ Lⁿ for large n. Partial sums form a Cauchy sequence since Σ from m₁ to m₂ of Lⁿ is bounded by L^(N+1)/(1−L) < ε.
Def 5.31 Cauchy Problem (ODE IVP)
Data: (x₀,y₀) ∈ ℝ², a,b > 0, and continuous f : [x₀−a, x₀+a]×[y₀−b, y₀+b] → ℝ.
y'(x) = f(x, y(x)) y(x₀) = y₀
Local solution: differentiable y : [x₀−α, x₀+α] → [y₀−b, y₀+b] satisfying both conditions, for some 0 < α ≤ a.
Lemma 5.36. y solves the Cauchy problem ⟺ y solves the Integral Equation (IECP): y(x) = y₀ + ∫_{x₀}^{x} f(t,y(t))dt
Thm 5.32 / 5.38 Peano–Picard Theorem
Statement. If ∂f/∂y exists and is continuous near (x₀,y₀), then the Cauchy problem has a unique local solution.
Proof strategy (Thm 5.38). Set M = max|f|, L = max|∂f/∂y| on R. For h < min{a, b/M, 1/L}:
  • Step 1: X = {continuous y : [x₀−h,x₀+h]→[y₀−b,y₀+b]} is complete (closed in C([−h,h]))
  • Step 2: F(y)(x) = y₀ + ∫f(t,y(t))dt maps X→X since |F(y)(x)−y₀| ≤ Mα ≤ b
  • Step 3: F is a contraction: dL∞(F(y₁),F(y₂)) ≤ αL·dL∞(y₁,y₂), and αL < 1
  • CMT: Unique fixed point = unique local solution
Ex 5.41 Failure of Uniqueness
Example. y'(x) = 3y(x)^(2/3), y(x₀) = y₀.
∂f/∂y = 2y^(−1/3) → undefined at y=0 For y₀ ≠ 0: unique solution y(x) = (x−x₀ + y₀^(1/3))³ For y₀ = 0: infinitely many solutions: y≡0 and y(x)=(x−x₀)³ and more: yc,d = 0 on [−c,d], (x+c)³ left, (x−d)³ right

The Lipschitz condition (∂f/∂y bounded) fails at y₀=0, so Peano–Picard does not apply and uniqueness breaks down.

Remark 5.40 Picard Iteration
Algorithm. Start with any u ∈ X. Iterate:
u, F(u), F²(u), F³(u), … Fⁿ(u) →_{dL∞} solution of C.P.
Why it works. F is a contraction on complete X, so by CMT the iterates form a Cauchy sequence converging to the unique fixed point.
  • Gives a constructive method to approximate solutions
  • Rate of convergence: dL∞(Fⁿ(u), y*) ≤ Lⁿ/(1−L) · dL∞(u, F(u))

Common Methods
Step-by-step solution guides for exam question types
Type 1 ℓᵖ Membership Classification

Given a sequence (aₙ), determine which of ℓ¹, ℓ², ℓ∞ it belongs to.

  • Step 1 — ℓ∞: Is the sequence bounded? If sup|aₙ| < ∞, it's in ℓ∞. If terms don't tend to zero or grow, it's NOT in ℓ∞.
  • Step 2 — ℓ¹: Does Σ|aₙ| converge? Use comparison, p-series (Σ1/nᵖ converges iff p>1), or geometric series. If it diverges, not in ℓ¹.
  • Step 3 — ℓ²: Does Σaₙ² converge? Same tests. Note ℓ¹ ⊆ ℓ² ⊆ ℓ∞, so if ∈ ℓ¹ then ∈ ℓ².
  • Step 4 — Conclude: State clearly which spaces it belongs to and which it doesn't, with one-line justification each.
Key facts: ℓ¹ ⊆ ℓ² ⊆ ℓ∞ (inclusions are strict) Σ1/n diverges → (1/n) ∉ ℓ¹ Σ1/n² converges → (1/n) ∈ ℓ² Constant nonzero sequence ∈ ℓ∞ only
Type 2 Convergence in C[0,1]: dL∞ vs dL1

Prove or disprove that a sequence fₙ converges to f under dL∞ or dL1.

  • For dL∞ — divergence: Find a single point x* where fₙ(x*) ↛ f(x*), or show sup|fₙ−f| ≥ c > 0 for all n. Often x*=1 works for fₙ=xⁿ.
  • For dL∞ — convergence: Show sup_{x∈[0,1]}|fₙ(x)−f(x)| → 0 explicitly (find the max via calculus if needed).
  • For dL1 — convergence: Compute ∫₀¹|fₙ(x)−f(x)|dx directly, e.g. ∫₀¹xⁿdx = 1/(n+1) → 0.
  • For dL1 — divergence: Show the integral stays bounded away from 0.
  • Key warning: If the pointwise limit is discontinuous, the sequence cannot converge in dL∞ (uniform limit of continuous functions is continuous).
Type 3 Open / Closed Set Classification

Given (X,d) and A ⊆ X, decide if A is open, closed, both, or neither.

  • Step 1 — Identify what A looks like. Is it a ball, a whole space, defined by ≤ or <, a finite set?
  • Open check: For every x ∈ A, can you find r>0 with B_r(x) ⊆ A? If A = X, always open. If defined by strict inequality (<), usually open.
  • Closed check: Is the complement Aᶜ open? Equivalently, does every convergent sequence in A have its limit in A? If defined by ≤, usually closed.
  • Finite sets in (ℝ,d₁): closed (singletons are closed), not open (every ball contains other points).
  • A = X: both open and closed always.
  • In C[0,1] with dL∞: sets defined by max|f−g| ≤ r are closed balls — closed. Strict < gives open balls — open.
Template answer structure: "A is [open/not open] because [for each x∈A, B_r(x)⊆A / there exists x∈A with no such r]." "A is [closed/not closed] because [Aᶜ is open / sequence xₙ→x with xₙ∈A but x∉A]."
Type 4 Proving a Product Metric

Given D on X×Y (or X×Y×Z), verify all metric axioms.

  • (M0) Non-negativity: D ≥ 0 because it's a sum/max of non-negative terms (each component metric is ≥ 0).
  • (M1) Identity: D(p,q)=0 iff each component dX(x₁,x₂)=0 AND dY(y₁,y₂)=0 iff x₁=x₂ AND y₁=y₂ iff p=q. (For max: max=0 iff all terms=0.)
  • (M2) Symmetry: D(p,q)=D(q,p) follows immediately from symmetry of each component metric.
  • (M3) Triangle inequality: Use triangle inequality component-wise. For D₁ (sum): add the inequalities dX(x₁,x₃)≤dX(x₁,x₂)+dX(x₂,x₃) and same for Y,Z. For D∞ (max): use max{a+b, c+d} ≤ max{a,c}+max{b,d}.
  • Conclude: "Hence all axioms (M0)–(M3) are satisfied, so D is a metric on X×Y×Z."
Type 5 Computing dL∞ and dL1 Between Functions

Given f,g ∈ C[0,1], compute dL∞(f,g) = max|f−g| and dL1(f,g) = ∫|f−g|.

  • Step 1 — Simplify h(x) = f(x)−g(x). Often terms cancel nicely.
  • Step 2 — dL∞: find the maximum of |h(x)| on [0,1]. Compute h'(x), find interior critical points, evaluate |h| at each critical point and at x=0,1. Take the largest value.
  • Step 3 — dL1: find zeros of h(x) (where f and g cross), split [0,1] into subintervals, integrate h on each piece, and sum the absolute values.
  • If h has no zeros: just compute ∫₀¹|h(x)|dx = |∫₀¹h(x)dx| directly.
Useful integrals: ∫₀¹ xⁿ dx = 1/(n+1) ∫₀¹ xⁿ(1−x) dx = 1/((n+1)(n+2))
Type 6 Transformed Metrics: is ef(x,y) = f(d(x,y)) a Metric?

Given a function f : ℝ≥0 → ℝ≥0, decide if ef(x,y) = f(d(x,y)) defines a metric.

  • Step 1 — Check (M0): is f(t) ≥ 0 for all t ≥ 0? If f can be negative (e.g. f(t)=3t−1 gives f(0)=−1), it fails immediately.
  • Step 2 — Check (M1): is f(t)=0 iff t=0? f(0) must equal 0. If f(0)≠0, fails.
  • Step 3 — Check (M2): f(d(x,y))=f(d(y,x)) — automatic since d is symmetric.
  • Step 4 — Check (M3): triangle inequality. Need f(d(x,z)) ≤ f(d(x,y))+f(d(y,z)) whenever d(x,z) ≤ d(x,y)+d(y,z). If f is increasing and subadditive (f(a+b)≤f(a)+f(b)), this works. Otherwise find a counterexample.
  • Counterexample strategy: Use (X,d)=(ℝ,d₁). Pick x=0, y=1, z=2 and check if M3 fails numerically.
Type 7 Computing d∞ Between Bounded Sequences

Given (aₙ) and (bₙ) in ℓ∞, compute d∞ = sup_n |aₙ−bₙ|.

  • Step 1: Write out |aₙ−bₙ| as a formula. Simplify.
  • Step 2: If the sequence has a pattern (e.g. even/odd terms behave differently), split into cases.
  • Step 3: Determine if |aₙ−bₙ| is increasing or decreasing. Check a few small values of n and the limit as n→∞.
  • Step 4: The supremum is either achieved at a specific n (state which) or is a limit (e.g. sup n/(n+1) = 1, not attained).
Common suprema: sup n/(n+1) = 1 (not attained) sup (1−1/n) = 1 (not attained) sup e⁻ⁿ = 1 (attained at n=0)
Type 8 Finding Isometries

Given metric spaces A and B, find φ : A → B that is surjective and distance-preserving.

  • In ℝ¹: isometries are φ(x)=x+a (translation) or φ(x)=a−x (reflection). Choose a so φ maps A onto B.
  • In ℝ²: use translations, rotations, or reflections. Sketch A and B — identify which rigid motion aligns them.
  • In C[0,1] with dL∞: translation by a fixed g is always an isometry: φ(f)=f+g. Also φ(f)(x)=f(1−x) or φ(f)=−f work.
  • In ℓᵖ spaces: try coordinate shifts/reflections, e.g. (a₀,a₁,…)↦(−a₀+c, a₂, a₃,…).
  • To verify: (1) show φ maps A into B, (2) exhibit a preimage for any b∈B (surjectivity), (3) show dY(φ(a),φ(b))=dX(a,b).
  • Non-isometry proof: find a configuration (e.g. equilateral triangle) possible in one space but not the other.
Type 9 Open/Closed Ball Membership

Given (X,d), centre x₀, radius r — decide if point p is in B_r(x₀) and/or B̄_r(x₀).

  • Step 1: Compute δ = d(p, x₀) using the given metric.
  • δ < r: p ∈ open ball and closed ball.
  • δ = r: p ∈ closed ball only.
  • δ > r: p in neither.
Distance formulas: d₂: √(Σ(xᵢ−yᵢ)²) d₁: Σ|xᵢ−yᵢ| d∞: max|xᵢ−yᵢ| d_disc: 0 if x=y, else 1 dL∞: max_{[0,1]}|f(x)−g(x)|
Type 10 Inequalities Between dₚ Metrics

Prove ordering relations between d₁, dₚ, d∞ on ℝⁿ.

  • d∞ ≤ dₚ: the max term alone contributes at least d∞ᵖ to the sum, so dₚ ≥ d∞.
  • d₁ ≤ n·d∞: bound each of the n terms in Σ|xᵢ−yᵢ| by d∞, then sum.
  • dₚ ≤ dq (p≥q): bound |xᵢ−yᵢ|ᵖ ≤ dq^(p−q)·|xᵢ−yᵢ|q, sum, take roots.
  • Equivalence: all dₚ on ℝⁿ are equivalent — bounded above and below by multiples of each other.
d∞ ≤ d₂ ≤ d₁ ≤ n·d∞ (on ℝⁿ) d∞ ≤ d₂ ≤ √n·d∞ (tighter via C-S)
