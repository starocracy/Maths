If p is prime and p | ab, then p | a or p | b (or both).

§1 — The Integers
The Norm & Fundamental Theorem
The norm function on ℤ and the Fundamental Theorem of Arithmetic.
Definition — The Norm

The norm N : ℤ → ℤ₊ is defined by N(a) = |a|.

Theorem
N(ab) = N(a) · N(b)
Theorem
N(a) = 0 ⟺ a = 0

Theorem — Every non-unit factorises
Every a ∈ ℤ, a ≠ 0, ±1, is a product of irreducibles.
Proof Sketch

If ±1, ±a are the only divisors of a → a is irreducible. Done.

Otherwise a = bc with b,c not units, so N(b) < N(a) and N(c) < N(a). Apply induction on N(a). ∎

Fundamental Theorem of Arithmetic
Theorem (Main Theorem of Arithmetic)

Every a ≠ 0, ±1 has a unique factorisation:

a = ε · p₁^m₁ · p₂^m₂ · ... · pₙ^mₙ

where ε = ±1, the pᵢ are distinct primes with p₁ < p₂ < ... < pₙ, and mᵢ ≥ 1.

Uniqueness: If also a = ε'q₁···qₘ then ε' = ε, n = m, and pᵢ = q_{σ(i)} for some bijection σ.

Finding gcd via canonical form
Write both numbers in canonical form, then select common primes raised to their minimal power.
§1 — The Integers
Euclid's Algorithm
Division with remainder and the algorithmic computation of gcd.
Theorem — Division with Remainder
For a, b ∈ ℤ, b ≠ 0: there exist unique q, r ∈ ℤ such that
a = q · b + r and N(r) < N(b)
Hint: Consider a/b and take q ∈ ℤ with |q − a/b| < 1.
The Algorithm

Given a,b ∈ ℤ\{0} with N(a) ≥ N(b), set a₁=a, a₂=b and iterate:

a₁ = q₁·a₂ + a₃ [N(a₂) > N(a₃)] a₂ = q₂·a₃ + a₄ [N(a₃) > N(a₄)] ⋮ aₖ = qₖ·aₖ₊₁ [remainder = 0 next step] ⟹ aₖ₊₁ = gcd(a₁, a₂)
Bézout's Identity
Back-substituting from the algorithm expresses:
gcd(a,b) = u·a + v·b for some u, v ∈ ℤ
In practice, use matrix row-reduction to track u and v simultaneously.
Worked Example: a = 15, b = 4
Matrix Method
(1 0 | 15) →R₁−3R₂→ (1 −3 | 3) →R₂−R₁→ (0 1 | 4) (0 1 | 4) → (1 −3 | 3) →R₁−3R₂→ (* * | 0) (−1 4 | 1) (−1 4 | 1) Result: 1 = (−1)·15 + 4·4 = gcd(4, 15)

Division & Factorisation in ℤ[i]
Euclidean division and unique factorisation in ℤ[i].
Construction — Division with Remainder

Given z, w ∈ ℤ[i] with N(z) ≥ N(w) ≥ 1:

  1. Compute z/w = α + βi ∈ ℂ as complex numbers.
  2. Round: take α₀, β₀ ∈ ℤ with |α−α₀| ≤ ½ and |β−β₀| ≤ ½.
  3. Set q = α₀ + β₀i ∈ ℤ[i]. Then z = qw + r with N(r) < N(w).
Worked Example
z = 15+8i, w = 1−3i
z/w = (15+8i)/(1−3i) · (1+3i)/(1+3i) = −9/10 + 53/10 · i Round: α₀ = −1, β₀ = 5 → q = −1 + 5i Check: 15+8i = (−1+5i)(1−3i) + 1 ✓ N(1) = 1 < N(1−3i) = 10 ✓
Theorem — Unique Factorisation in ℤ[i]

ℤ[i] has Euclid's Algorithm, hence Unique Factorisation.

Every z ∈ ℤ[i], z ≠ 0, not a unit, writes as:

z = ε · p₁ · p₂ · ... · pₙ (ε a unit, pᵢ irreducible)

Uniqueness: if also z = ε'q₁···qₘ then m=n and qᵢ = εᵢ·p_{σ(i)} for units εᵢ and bijection σ.

Rings Similar to ℤ[i]
Comparison of ℤ[√d] for various d, and which have unique factorisation.
RingNormUnitsUFD?Key Test

The Key Condition
Division with remainder works (giving Euclid's Algorithm) when rounding to the nearest lattice point leaves a remainder with norm strictly less than the divisor's norm. The test quantity must be < 1.
Counterexample — ℤ[√−3] fails
4 = 2 · 2 = (1 + √−3)(1 − √−3)

These are two genuinely different factorisations — unique factorisation fails in ℤ[√−3].

The reason: (½)² + 3(½)² = 1, which is not strictly less than 1, so the Euclidean algorithm doesn't work here.

Conclusion for §2
ℤ[i], ℤ[√2], ℤ[√−2] all have Euclid's Algorithm ⟹ Unique Factorisation and hence share the same arithmetic properties as ℤ.

§3 — Polynomials
GCD & Unique Factorisation
Greatest common divisors and unique factorisation for polynomials.
Theorem — Unique Factorisation for Polynomials

Every f(x) with deg f ≥ 1 is a product of irreducibles:

f = p₁ · p₂ · ... · pₙ (each pᵢ irreducible, deg pᵢ ≥ 1)

If also f = q₁···qₘ (irreducible), then n = m and qᵢ = εᵢ · p_{σ(i)} for constants εᵢ ∈ K and bijection σ.

Greatest Common Divisor
Definition — gcd of polynomials

d(x) is a gcd of f, g ∈ K[x] iff:

  • d | f and d | g
  • For any d₁ with d₁|f and d₁|g: d₁ | d

gcd is unique up to a non-zero constant. Made canonical by requiring it to be monic (leading coefficient = 1).

Theorem — Bézout for Polynomials
For some u(x), v(x) ∈ K[x]:
d(x) = u(x)·f(x) + v(x)·g(x)
Computed via Euclid's Algorithm using matrix row-reduction.
Worked Example
f = x³+x²+x+1, g = x²−1
① deg f > deg g, divide: x³+x²+x+1 = (x+1)(x²−1) + 2x+2 ② deg(x²−1) > deg(x+1), divide: x²−1 = (x−1)(x+1) + 0 ⟹ gcd = x+1 (monic) Bézout: (x+1) = (½)·f + (−½x−½)·g
Chapter I — Core Conclusion
Euclid's Algorithm ⟹ Unique Factorisation
This holds in ℤ, ℤ[i], ℤ[√2], ℤ[√−2], and K[x]. All are Euclidean Domains, hence UFDs.
Number-Theoretic Application
Fermat's Equation: x³ = y² + 2
Proving (3, ±5) are the only integer solutions, using ℤ[√−2].
Problem (P. Fermat)
Find all (x,y) ∈ ℤ × ℤ satisfying x³ = y² + 2.

Known solutions: (x,y) = (3, ±5). Are these the only ones?

Proof Strategy
  1. Work in ℤ[√−2]: write x³ = (y+√−2)(y−√−2).
  2. Show the two factors are coprime: gcd(y+√−2, y−√−2) = 1.
  3. Since ℤ[√−2] is a UFD and x³ is a cube, each factor must itself be a cube.
  4. Set y+√−2 = (a+b√−2)³ and expand.
  5. Read off the coefficient of √−2 to get a constraint on b.
  6. Case-check b = ±1 to find all solutions.
Key Lemma
gcd(y+√−2, y−√−2) = 1 in ℤ[√−2].
Proof of Lemma

Suppose irreducible d ∈ ℤ[√−2] (not a unit) divides both. Then d | 2√−2 = −(√−2)³.

But √−2 is irreducible (its norm is 2, a prime). So d = √−2 in ℤ[√−2] (up to units).

Then √−2 | y+√−2, so y = −2b and a=1 for some integers. Back in original equation: x³ = 4b²+2, so x is even (x=2ℓ), giving 4ℓ³ = 2b²+1 — impossible (LHS even, RHS odd). ∎

Expanding the Cube
y + √−2 = (a + b√−2)³ = (a³ − 6ab²) + (3a² − 2b²)b · √−2 ↑ This must equal 1 So: b(3a² − 2b²) = 1 ⟹ b = ±1
CaseEquationSolutions
b = 1 3a² − 2 = 1 ⟹ a = ±1 y = a³−6ab² = ±5, x = 3 ✓
b = −1 3a² − 2 = −1 ⟹ 3a² = 1 Impossible in ℤ ✗
Conclusion
The only integer solutions to x³ = y² + 2 are (x, y) = (3, 5) and (x, y) = (3, −5).

Definition of an abelian group and notation for the operation.
Definition — Abelian (Commutative) Group

A set A equipped with a binary operation A × A → A, (a,b) ↦ a∗b, satisfying:

AxiomNameStatement
(A1)Associativity(a∗b)∗c = a∗(b∗c) for all a,b,c

Chapter II §1 — Groups
Subgroups & Cosets
Subgroups, H-cosets, and the coset partition theorem.
Definition — Subgroup

B ⊂ A is a subgroup if for any v, u ∈ B, the element v − u ∈ B as well.

In particular: 0A ∈ B (additive), or 1A ∈ B (multiplicative), and vu⁻¹ ∈ B.

Theorem — Subgroups of ℤ

Every subgroup H ⊂ ℤ is of the form H = dℤ = {0, ±d, ±2d, …} for some d ≥ 0.

Proof sketch: If H ≠ {0}, take d ∈ H\{0} with |d| minimal. For any m ∈ H, write m = qd + r with |r| < |d|, so r ∈ H forces r = 0, giving m ∈ dℤ. ∎

H-Cosets
Definition — H-Coset
For A a group, H ⊂ A a subgroup, a ∈ A any element:
a + H = {a + h | h ∈ H} is an H-coset
Proposition — Cosets are disjoint or equal

Cosets a+H and b+H either are disjoint ({a+H} ∩ {b+H} = ∅) or coincide (a+H = b+H).

The latter occurs if and only if a − b ∈ H.

Proof: If not disjoint: a+h₁ = b+h₂ for some h₁,h₂ ∈ H, so a−b = h₂−h₁ ∈ H, then a+H = b+(a−b)+H = b+(h₂−h₁)+H = b+H. ∎

Chapter II §1 — Groups
Quotient Groups & Isomorphisms
The quotient group construction and the notion of isomorphism.
Definition — Quotient Group A/H

The set of H-cosets {a+H}, equipped with the operation:

{a+H} + {b+H} = (a+b)+H

forms a group: the quotient group A/H, with zero 0A/H = {H}.

Well-defined: if a₁ = a+h₁ ∈ a+H and b₁ = b+h₂ ∈ b+H, then a₁+b₁ ∈ a+b+(h₁+h₂)+H = (a+b)+H. ✓

Key Example — ℤ/nℤ
A = ℤ, H = nℤ: ℤ/nℤ = {nℤ} ∪ {1+nℤ} ∪ … ∪ {(n−1)+nℤ} ̄0 ̄1 (n̄−1) = the group of congruence classes mod n ̄i + ̄j = i+j mod n

ℤ/(0) = ℤ; A/(0) = A always.

Def: ℤ/nℤ is the additive cyclic group of order n.

Isomorphisms
Definition — Isomorphism

A bijection φ: A → B is an isomorphism if:

φ(a₁ ∗ a₂) = φ(a₁) ∗ φ(a₂) for all a₁,a₂ ∈ A

It follows: φ(eA) = eB and φ(a⁻¹) = φ(a)⁻¹.

If such φ exists, write A ≅ B (A and B are isomorphic).

Example — Cyclic group Cₙ

Let ζ = e^(2πi/n) ∈ ℂ, so ζⁿ = 1. The set Cₙ = {1, ζ, …, ζⁿ⁻¹} is a multiplicative group.

The map φ: ℤ/nℤ → Cₙ, i ↦ ζ�ⁱ is an isomorphism (of additive ↔ multiplicative cyclic groups).

Chapter II §1 — Groups
Order & Lagrange's Theorem
Order of a group, order of an element, and Lagrange's theorem.
Definition — Order of a Group

|A| = number of elements in A. Can be ∞ or a positive integer.

|Cₙ| = n, |ℤ| = ∞

Definition — Order of an Element

ord(a) = least positive k such that:

ka = 0 (additive) aᵏ = 1 (multiplicative)

Theorem (Lagrange — special case)

If |A| < ∞ and a ∈ A, then ord(a) divides |A|.

Proof sketch (additive): List all n = |A| elements a₁,…,aₙ. The shifts bᵢ = aᵢ+a are the same n elements in some order. So Σbᵢ = Σaᵢ, giving na + Σaᵢ = Σaᵢ, hence na = 0. ∎

Order in Cyclic Groups
Theorem — Order of aᵏ in Cₙ

In Cₙ = {1, a, …, aⁿ⁻¹} with aⁿ = 1:

ord(aᵏ) = n / gcd(k,n)

In particular: ord(aᵏ) = n ⟺ gcd(k,n) = 1 (i.e. k and n are coprime).

If (k,n) = 1, then b = aᵏ is a generator of Cₙ: Cₙ = {1, b, b², …, bⁿ⁻¹}.

Direct Products & Direct Sums
Combining groups via direct products/sums and order in products.
Direct Product A₁ × A₂

Set: {(a₁,a₂) | a₁ ∈ A₁, a₂ ∈ A₂}

Operation: (a₁,a₂)∗(b₁,b₂) = (a₁b₁, a₂b₂)

Identity: 1A₁×A₂ = (1A₁, 1A₂)

Inverse: (a₁,a₂)⁻¹ = (a₁⁻¹, a₂⁻¹)

Direct Sum A₁ ⊕ A₂

Same as direct product with additive notation:

(a₁,a₂) + (b₁,b₂) = (a₁+b₁, a₂+b₂)

Zero: 0 = (0,0)

Example: ℤᵏ = ℤ⊕…⊕ℤ (k times)

Note — Sizes
If A₁, A₂ finite: |A₁ × A₂| = |A₁| · |A₂| and |A₁ ⊕ A₂| = |A₁| · |A₂| (not sum!)
Theorem — Order in A₁ × A₂

For a = (a₁,a₂) ∈ A₁ × A₂:

  • (i) If ord(aᵢ) = ∞ for at least one i, then ord(a) = ∞.
  • (ii) If ord(a₁) = n₁ and ord(a₂) = n₂, then ord(a) = lcm(n₁,n₂).

Proof of (ii): aᵏ = 1 iff both a₁ᵏ = 1 and a₂ᵏ = 1, iff n₁|k and n₂|k, i.e. k is a multiple of lcm(n₁,n₂). ∎

Corollary — When is Cₙ × Cₘ cyclic?
gcd(n,m) = 1 ⟹ Cₙ × Cₘ ≅ Cₙₘ

If gcd(n,m) > 1, then Cₙ × Cₘ is NOT cyclic — no element has order nm.

Examples: C₃₆ ≅ C₄ × C₉, but C₃₆ ≇ C₃ × C₁₂ (since gcd(3,12)=3 > 1).

Chapter II §2 — Direct Products
Cyclic Groups & Decomposition
The infinite cyclic group C∞, primary decomposition of cyclic groups, rank and torsion.
Definition — C∞ (Infinite cyclic group)
C∞ = {1, a, a⁻¹, a², a⁻², …} = {aⁱ | i ∈ ℤ} aⁱ⁺ʲ = aⁱ · aʲ

The map ℤ → C∞, i ↦ aⁱ is an isomorphism (additive ↔ multiplicative).

Primary Decomposition of Cₙ

Write n = p₁^α₁ · … · pₖ^αₖ (distinct primes). Then:

Cₙ ≅ Cₙ₁ × … × Cₙₖ where nᵢ = pᵢ^αᵢ

(The primary decomposition — each Cₙᵢ cannot be decomposed further.)

Example: C₇₂ ≅ C₈ × C₉ = C₇₂^(2) × C₇₂^(3)

Rank and Torsion
Definitions — Rank & Torsion

Any abelian group we know has the form:

A ≅ C∞ × … × C∞ × Cₘ₁ × … × Cₘₗ r times
  • rk(A) = r is the rank of A.
  • torsion(A) = Cₘ₁ × … × Cₘₗ is the torsion of A.

So A ≅ C∞^{rk(A)} × torsion(A).

Chapter II §2 — Direct Products
Primary Decomposition & Smith Normal Form
Classification of finite abelian groups and the Smith Normal Form algorithm.
Theorem — Finite Abelian Groups
Every finite abelian group has a primary decomposition (of the form Cₙ₁ × … × Cₙₖ with each nᵢ a prime power) which is unique up to permuting the factors.
Corollary — When are two groups isomorphic?
A ≅ B ⟺ rk(A) = rk(B) AND torsion(A) and torsion(B) have the same primary decomposition
Smith Normal Form (SNF)
Construction — SNF

Starting from the primary decomposition, use the rule Cₙ × Cₘ ≅ Cₙₘ (when gcd(n,m) = 1) to collect all max powers of all primes into one factor; split it off and repeat.

Result: a presentation Cₐ₁ × Cₐ₂ × … with a₁ | a₂ | a₃ | … in decreasing magnitude order.

Worked Example: SNF(C₂₈ × C₉₈)
Primary decomp: C₂₈ × C₉₈ = C₄×C₇×C₂×C₄₉ ≅ C₂×C₄ × C₇×C₄₉ (prime 2) (prime 7) Group columns by prime, then take MAX power per column: Col 1 (largest): C₄₉ × C₄ → C₁₉₆ Col 2 (next): C₇ × C₂ → C₁₄ SNF(C₂₈ × C₉₈) = C₁₉₆ × C₁₄
Important Restrictions on Row/Column Operations

Allowed: R₁ ↔ R₂, Rᵢ + c·Rⱼ (c ∈ ℤ, j≠i), (−1)Rᵢ, and column analogues.

NOT allowed: ⅓R₂, 3Cⱼ — fractions or scaling by |c| > 1 are NOT valid integer row operations!

Do one operation at a time — easy to make mistakes combining steps.

Chapter II §3 — Classification
Classification of Finitely Generated Abelian Groups
Generators, relations, the matrix method, and the main classification theorem.
Definition — Generators

a₁,…,aₙ ∈ A generate A if every a ∈ A can be written as:

a = m₁a₁ + … + mₙaₙ for some m₁,…,mₙ ∈ ℤ

A is finitely generated if it has a finite generating set.

Examples: (ℚ,+) is NOT finitely generated; (ℤ,+) is generated by +1 or −1; Cₙ is generated by any aᵏ with gcd(k,n) = 1.

Theorem — Classification of f.g. Abelian Groups

Any finitely generated abelian group A is isomorphic to:

A ≅ C∞^r × Cₙ₁ × … × Cₙₖ

for some r ≥ 0 and positive integers n₁,…,nₖ. This can be expressed as a primary decomposition (then reassembled into SNF).

Proof Strategy via Relations Matrix
  1. Let g₁,…,gₙ generate A. A relation is (a₁,…,aₙ) ∈ ℤⁿ with a₁g₁+…+aₙgₙ = 0. These form a subgroup H ⊂ ℤⁿ.
  2. Show A ≅ ℤⁿ/H via the map π: ℤⁿ/H → A sending m̲+H ↦ m₁g₁+…+mₙgₙ (well-defined since relations map to 0; bijective since g₁,…,gₙ generate A).
  3. H is generated by ≤ n relations; organise as an (m×n) matrix Q with integer entries.
  4. Apply integral row ops (R1: Rᵢ↔Rⱼ; R2: Rᵢ+cRⱼ) and column ops (C1: Cᵢ↔Cⱼ; C2: Cᵢ+cCⱼ) to diagonalise Q.

Theorem — Chinese Remainder Theorem (CRT) for ℤ

If gcd(m,n) = 1:

ℤ/(mn) ≅ ℤ/(m) ⊕ ℤ/(n)

via φ: a+(mn) ↦ (a+(m), a+(n))

Proof sketch

φ⁻¹(b+(m), c+(n)) = cum + bvn + (mn)

Check: cum+bvn ≡ b·vn ≡ b(1−um) ≡ b mod m. ∎

An ideal M ⊂ R, M ≠ R, is maximal if there is no ideal N ≠ R with M ⊊ N (nothing between M and R).

Theorem — Maximal ⟺ Quotient is a Field
M ⊂ R maximal ⟺ R/M is a field

Proof (⟹): Take a∈R\M. The ideal (a)+M = R, so a·b+m = 1R for some b∈R, m∈M. Then ā·b̄ = 1 in R/M. ∎

Proof (⟸): Take a∈R\M; ā=a+M is invertible in R/M, so ā·b̄ = 1R/M, so ab = 1R+m for some m∈M. Any ideal N containing M and a then contains 1R, so N=R. ∎

UFDs & Gauss's Theorem
Unique Factorisation Domains (general rings), Gauss's Lemma, and the content of a polynomial.
Definition — Irreducible (in general ring)
a∈R is irreducible if a=bc implies b or c is a unit. Otherwise reducible.
Definition — UFD
R (no ZDs) is a UFD if every r∈R, r≠0 and not a unit, writes as r=a₁…aₖ with aᵢ irreducible, uniquely up to units and permutation.

Note on Irreducibility — Context Matters
x²+1 ∈ ℚ[x] irreducible 3x²+3 ∈ ℚ[x] irreducible (1/3 is a unit in ℚ) 3x²+3 ∈ ℤ[x] reducible (= 3·(x²+1), and 3 is not a unit in ℤ) x²−1 ∈ ℚ[x] reducible (= (x+1)(x−1))

Any field is a UFD: no irreducible elements at all (every a=0 or a is a unit).

Definition — Content of a Polynomial
For f(x) = aₙxⁿ+…+a₀ ∈ ℤ[x]:
cont(f) = gcd(a₀, a₁, …, aₙ) ∈ ℤ₊
Gauss's Lemma
For f,g ∈ ℤ[x]:
cont(fg) = cont(f) · cont(g)

Proof idea: WLOG cont(f)=cont(g)=1. For any prime p, take i = smallest index with p∤aᵢ, j = smallest with p∤bⱼ. The coefficient of x^{i+j} in fg is aᵢbⱼ + (terms with smaller aₛ or bₜ), all divisible by p except aᵢbⱼ. So p∤(that coefficient), hence cont(fg)=1. ∎

Theorem (Gauss) — ℤ[x] is a UFD

Key steps: Write f=cont(f)·f̄ with cont(f̄)=1. Factor f̄ in ℚ[x] (which is a UFD), then use Gauss's lemma to show the factors are in ℤ[x]. Uniqueness follows since ℚ[x] is a UFD and units of ℤ are ±1. ∎

Corollary — Irreducibility transfers
f(x) ∈ ℤ[x] irreducible in ℤ[x] ⟹ f(x) irreducible in ℚ[x]
R a UFD ⟹ R[x] is a UFD
In particular: K[x₁,…,xₙ] is a UFD for any field K

Also: f,g ∈ ℤ[x] with cont(f)=cont(g)=1 and f|g in ℚ[x] ⟹ f|g in ℤ[x].

And: monic f ∈ ℤ[x] with rational root α ∈ ℚ ⟹ α ∈ ℤ.

Theorem — Eisenstein's Criterion

Let f(x) = aₙxⁿ+…+a₁x+a₀ ∈ ℤ[x] with cont(f)=1. Suppose for some prime p:

  • p ∤ aₙ (does not divide leading coefficient)
  • p | aᵢ for i = n−1, n−2, …, 0 (divides all others)
  • p² ∤ a₀ (does not divide constant term twice)

Then f is irreducible in ℤ[x] (and in ℚ[x]).

Proof sketch
Assume f=gh. Since p∤aₙ: p∤bₖ and p∤cₗ (leading coefficients of g,h). Since a₀=b₀c₀ and p|a₀ but p²∤a₀: p divides exactly one of b₀,c₀; say p|b₀ but p∤c₀. Let j be smallest with p∤bⱼ; then j≤k < n. The coefficient aⱼ = bⱼc₀ + (terms bₛ with s<j, all divisible by p), so p∤aⱼ for some j<n — contradicting p|aⱼ for all j<n. ∎
Examples
Direct application (p=2)
xⁿ + 2xⁿ⁻¹ + 2x − 2 p=2: 2∤1 (leading), 2|others, 4∤(−2) ∴ irreducible ✓
Substitution trick
f(x) = x⁵+5x⁴+10x³+10x²+8x+7 f(x) = g(x+1), g(y) = y⁵+3y+3 g satisfies EC for p=3 ⟹ g irreducible ⟹ f(x) = g(x+1) irreducible ✓
Cyclotomic Polynomial Φₚ(t) is irreducible
Φₚ(t) = tᵖ⁻¹+tᵖ⁻²+…+t+1 = (tᵖ−1)/(t−1)

Consider Φₚ(x+1) = ((x+1)ᵖ−1)/x = xᵖ⁻¹ + (p choose p−1)xᵖ⁻² + … + (p choose 1).

All middle binomials (p choose i) for i=1,…,p−1 are divisible by p. Constant term (p choose 1) = p, not divisible by p². So EC applies with this prime p ⟹ Φₚ(x+1) irreducible ⟹ Φₚ(t) irreducible.

Application — Sum of Two Squares

An odd prime p = m²+n² ⟺ p ≡ 1 mod 4.

Proof (⟸): p ≡ 4k+1. In 𝔽ₚ, (−1) is a square (shown using Lagrange's theorem: b ∈ 𝔽ₚ* is a square iff b^{(p−1)/2}=1; and (−1)^{(p−1)/2}=(−1)^{2k}=1). So p | (a²+1) for some a∈ℤ, giving pq=(a+i)(a−i) in ℤ[i]. If p were irreducible in ℤ[i], then p|(a+i) — impossible since a/p+i/p ∉ ℤ[i]. So p is reducible: p=u·v, N(u)=N(v)=p. Writing u=m+ni gives p=m²+n². ∎

If a field K is a subset of a field L, we say L is an extension of K, K is a subfield of L: K ⊂ L.

Any finite extension K ⊂ L is algebraic.

Theorem — Tower Law (Multiplicativity of Degree)

If K ⊂ L is finite with basis e₁,…,eₙ and L ⊂ Q is finite with basis h₁,…,hₘ, then K ⊂ Q is finite with basis eᵢhⱼ (i=1,…,n; j=1,…,m). In particular:

[Q : K] = [Q : L] · [L : K]

Proof: Any α∈Q writes as α = Σᵢ bᵢhᵢ (bᵢ∈L), and each bᵢ = Σⱼ bᵢⱼeⱼ (bᵢⱼ∈K), giving α = Σᵢⱼ bᵢⱼ(eⱼhᵢ) — unique since both representations are unique. ∎

• If a₁=0: a₀²=5 — impossible (Eisenstein p=5 shows 3x²−5 irred. in ℚ[x]).

• If a₀=0: 3a₁²=5 — impossible.

So x²−5 is irreducible over ℚ(√3).

Conclusion
[ℚ(√3,√5) : ℚ(√3)] = 2 and [ℚ(√3,√5) : ℚ] = 2·2 = 4

A natural basis: 1, √3, √5, √15 (where √15 = √5·√3).

Example: Finding a Minimal Polynomial
Find f(x) ∈ ℚ[x] with f(√3−√5) = 0

Set α = √3−√5. Express 1, α, α², α³, α⁴ in the basis {1, √3, √5, √15}:

α⁰: (1, 0, 0, 0) α¹: (0, 1,−1, 0) α²: (8, 0, 0,−2) [α² = 8−2√15] α³: (0,18,−14, 0) [α³ = 18√3−14√5] α⁴: (124, 0, 0,−32) [α⁴ = 124−32√15]

Rows 1, 3, 5 (for α⁰, α², α⁴) satisfy R5 + (−16)R3 + 4R1 = 0, giving:

f(x) = x⁴ − 16x² + 4

Note: f is irreducible — the first 4 rows of the matrix are linearly independent, ruling out any factor of degree ≤ 3.

Computing Inverses in Composite Extensions
Find 1/(1−√3+√5+√15)

Method 1: Write 1/(1−√3+√5+√15) = x₀+x₁√3+x₂√5+x₃√15, multiply out and solve a 4×4 linear system.

Method 2 (conjugation): Successively eliminate surds by multiplying by conjugates:

Step 1: multiply by (1−√3)−(1+√3)√5 to clear √5 → denominator becomes −16−12√3 Step 2: multiply by −16+12√3 to clear √3 → denominator becomes −176

Method 2 is generally more computationally straightforward.

Chapter IV §1 — Fields
Constructing Fields by Adding a Root
How to extend a field K by adjoining a root of an irreducible polynomial via the quotient ring K[x]/(f).
Setup

Let K be a field which is not algebraically closed, f(x) ∈ K[x] an irreducible polynomial of degree n ≥ 2 (so f has no roots in K). Consider the quotient ring L = K[x]/(f) with projection π: K[x] → K[x]/(f).

Theorem — L = K[x]/(f) is a Field Extension

(i) L = K[x]/(f) is a field.

(ii) K ⊂ L is a finite extension with [L : K] = deg f = n.

(iii) α = π(x) is a root of f(x) in L.

(iv) 1, α, …, αⁿ⁻¹ form a basis of L over K.

Proof sketch

(i) L is a field: For β∈L\{0}, write β = π(g) where g∈K[x] is not divisible by f. Since f is irreducible: gcd(f,g)=1, so u(x)g(x)+v(x)f(x)=1 for some u,v∈K[x]. Applying π: π(u)·β = 1 in L, giving β⁻¹ = π(u). ∎

(ii) & (iv) Basis: Any β=π(g) with g = q(x)f(x)+r(x), deg r ≤ n−1. Then β = π(r) = r₀+r₁α+…+rₙ₋₁αⁿ⁻¹. This shows [L:K] ≤ n. If [L:K] < n, then 1,α,…,αⁿ⁻¹ would be lin. dependent, so some r(x) of degree ≤n−1 with r(α)=0 — but then f | r, impossible since deg r < n. So [L:K]=n. ∎

(iii): f(α) = f(π(x)) = π(f(x)) = 0. ∎

Worked Example: ℚ[x]/(x³−2x+2)
f(x) = x³−2x+2 (irreducible over ℚ by Eisenstein p=2)

Let α be a root of f (α∈ℂ). Then [ℚ(α):ℚ]=3, with basis {1, α, α²}.

Multiplication rules (using α³ = 2α−2):

α·α² = α³ = 2α−2 α²·α² = α⁴ = α·α³ = α(2α−2) = 2α²−2α

Finding (1−3α+α²)⁻¹: Solve (x₀+x₁α+x₂α²)(1−3α+α²)=1. Expanding and collecting basis coefficients gives the system:

x₀ − 2x₁ + 6x₂ = 1 −3x₀ + 3x₁ − 8x₂ = 0 x₀ − 3x₁ + 3x₂ = 0

Solution: (x₀,x₁,x₂) = (1/19)(−15, 1, 6), so

(1−3α+α²)⁻¹ = −15/19 + (1/19)α + (6/19)α²
Chapter IV §1 — Fields
Splitting Fields & Normal Extensions
Splitting fields for polynomials, normal (Galois) extensions, and the example of x³−2 over ℚ.
Definition — Splitting Field

Let K be any field and f(x) ∈ K[x] (not necessarily irreducible). A finite extension K ⊂ L is a splitting field for f if:

(i) f(x) = aₙ∏ᵢ(x−αᵢ) with all αᵢ ∈ L (f splits completely in L[x]).

(ii) L = K(α₁,…,αₙ) — every β ∈ L is a polynomial in the roots.

Note: If f already splits in L=K (all roots in K), then K itself is the splitting field, e.g. x²−4: splitting field is ℚ.

Definition — Normal (Galois) Extension

An algebraic extension K ⊂ L is normal (Galois) if for every irreducible poly f(x) ∈ K[x]:

(f has a root in L) ⟹ f splits completely in L

That is: one root of f in L implies all roots of f are in L.

Theorem — Splitting Fields are Normal

If L is the splitting field for some g(x) ∈ K[x], then K ⊂ L is a normal (Galois) extension.

Ex. K=ℚ, f(x)=x²−3, L=ℚ(√3) is the splitting field for f. So if an irreducible q(x) over ℚ has a root in ℚ(√3) ⟹ all roots are in ℚ(√3).

Example: Splitting Field for x³−2 over ℚ
The three roots: α₁ = ∛2, α₂ = ζ∛2, α₃ = ζ²∛2

where ζ = e^{2πi/3} = −½ + (√3/2)i is a primitive cube root of unity.

Step 1: K₁ = ℚ(∛2), [K₁:ℚ] = 3 (degree of x³−2), basis {1, α₁, α₁²}.

Step 2: Since α₁,α₂ ∈ K (the full splitting field), ζ = α₂/α₁ ∈ K. Note ℚ(ζ) = ℚ(√−3), corresponding to the irreducible x²+x+1, so [K₂:ℚ]=2.

Step 3: [K:ℚ] is divisible by both 2=[K₂:ℚ] and 3=[K₁:ℚ], so divisible by 6. Since ℚ(∛2,ζ) contains all roots and [ℚ(∛2,ζ):ℚ] ≤ 6, we conclude:

K = ℚ(∛2, ζ), [K:ℚ] = 6

Basis: 1, ∛2, ∛4, ζ, ζ∛2, ζ∛4.

Also: x²+x+1 remains irreducible over ℚ(∛2) (since 6 is divisible by [K₂:ℚ]=2).

Chapter IV §2 — Fields
Finite Fields
Characteristic, finite field structure, and the classification theorem 𝔽_{pᵐ}.
Definition — Characteristic

For a field F, consider 1, 1+1, 1+1+1, … Two cases:

  • All distinct (all non-zero): we say char F = 0. Then F ⊃ ℚ.
  • Some k smallest with 1+…+1 (k times) = 0: char F = k. Then F ⊃ 𝔽ₖ.
Theorem — Characteristic is Prime

The characteristic k (when non-zero) is always a prime p.

Proof: Suppose k=nm with n,m≥2. Then (1+…+1)ₙ·(1+…+1)ₘ = (1+…+1)ₖ = 0. Since F has no zero divisors, one factor is 0 — contradicting k being minimal. ∎

Examples: char ℚ = 0, char 𝔽ₚ = p. If char F=0 then F⊃ℚ; if char F=p then F⊃𝔽ₚ.

Theorem — Size of Finite Fields

If L ⊃ 𝔽ₚ is a finite extension with [L:𝔽ₚ]=m, then |L| = pᵐ.

Proof: In a basis e₁,…,eₘ of L over 𝔽ₚ, every α = a₁e₁+…+aₘeₘ with aᵢ∈𝔽ₚ (p choices each). ∎

Classification of Finite Fields

Consider the multiplicative group L* = L\{0} of order pᵐ−1. By Lagrange's theorem, every α ∈ L* satisfies α^{pᵐ−1} = 1, so every α ∈ L satisfies:

x^{pᵐ} − x = ∏_{α∈L} (x−α)

L is the splitting field of fₘ(x) = x^{pᵐ}−x. For every q=pᵐ there is one and only one field with q elements, denoted 𝔽_q (unique up to isomorphism).

Examples: Building Finite Fields
𝔽₉ = 𝔽₃(i), i² = −1

Over 𝔽₃={0,1,2}, the irreducible quadratics are x²+2x+2, x²+x+2, x²+1.

All define the same extension:

𝔽₉ = {a+bi | a,b ∈ 𝔽₃} Work as usual: (1+2i)/(1+i) = (1+2i)(1−i)/2 = (3+i)/2 = i/2 = 2i (as 2⁻¹ = 2 in 𝔽₃, since 2·2=4=1)
𝔽₂₅ = 𝔽₅(α), α²=2

x²+3 is irreducible over 𝔽₅={0,1,2,3,4} (no roots — check x=0,1,2,3,4).

𝔽₂₅ = 𝔽₅[x]/(x²+3) = 𝔽₅(α) α² = 2 (since −3 = 2 in 𝔽₅) 𝔽₂₅ = {a+bα | a,b ∈ 𝔽₅} (2+3α)/(1+2α) = (2+3α)(1−2α)/(1−4·2) = α/2 = 3α
𝔽₁₂₅ = 𝔽₅(α), α³ = 3α+4

x³+2x+1 ∈ 𝔽₅[x] is irreducible (check: no roots in 𝔽₅). So 𝔽₁₂₅ = 𝔽₅[x]/(x³+2x+1) = 𝔽₅(α) where α³=3α+4 (since −2=3, −1=4 in 𝔽₅).

𝔽₁₂₅ = {a₀ + a₁α + a₂α² | aᵢ ∈ 𝔽₅}

Note: Working with finite fields (especially small primes) is easier than ℚ — many properties can be checked by direct inspection, and arithmetic has no fractions! E.g. in 𝔽₇: 4⁻¹=2.

Positive Characteristic & the Frobenius
Theorem — Inseparability in Char p

Let char K = p > 0, f(x) = xᵖ−a ∈ K[x], and α a root in some extension. Then:

f(x) = (x−α)ᵖ

Proof: (x−α)ᵖ = xᵖ + (p choose i) terms − αᵖ. All binomial coefficients (p choose i) for 0 < i < p are divisible by p, hence = 0 in K. And (−α)ᵖ = (−1)ᵖαᵖ = −a (since p is odd or p=2 and +1=−1 in 𝔽₂). ∎

This shows f(x) = xᵖ−a has ALL roots equal — a single root α with multiplicity p.

Chapter IV §3 — Fields
Fields of Rational Functions
Rational function fields, algebraic dependence, and transcendence degree.
Definition — Field of Rational Functions

Let K be any field and t₁,…,tₙ independent variables. K[t₁,…,tₙ] is the ring of polynomials. The field of rational functions is:

K(t₁,…,tₙ) = { p(t*)/q(t*) | p,q ∈ K[t*], q≠0 }

where p(t*)/q(t*) = p₂/q₂ iff p·q₂ = p₂·q as polynomials.

Definition — Algebraic Dependence / Independence

Rational functions f₁,…,fₘ ∈ K(t*) are algebraically dependent if there exists a non-zero polynomial G(u₁,…,uₘ) ∈ K[u₁,…,uₘ] such that G(f₁,…,fₘ) ≡ 0 (identically zero). Otherwise they are algebraically independent.

Theorem — Any (n+1) Elements of K(t₁,…,tₙ) are Algebraically Dependent

Proof sketch (n=2): For f₁,f₂,f₃ ∈ K[t₁,t₂] polynomials, consider Φ: K[u₁,u₂,u₃] → K[t₁,t₂] sending g(u₁,u₂,u₃) ↦ g(f₁,f₂,f₃). If deg fᵢ ≤ d and deg g ≤ N, then deg Φ(g) ≤ Nd. Comparing dimensions:

dim{deg≤N in 3 vars} = C(N+3,3) ~ N³/6 dim{deg≤Nd in 2 vars} = C(Nd+2,2) ~ N²d²/2

For large N, the source has higher dimension than the target, so ker Φ ≠ {0} — some non-zero polynomial g with g(f₁,f₂,f₃) ≡ 0. ∎

Definition — Transcendence Degree

For a field extension K ⊂ L, the transcendence degree tr.deg(L/K) is the maximal number of algebraically independent elements of L over K.

Ex: L = K(t₁,…,tₙ) ⟹ tr.deg(L/K) = n
(since t₁,…,tₙ are alg. independent over K)
Chapter IV §3 — Fields
Differentiation & Multiple Roots
Formal differentiation over any field, the Leibniz rule, and detecting multiple roots via gcd(f, f′).
Definition — Formal Derivative

For any field K, define the formal derivative d/dt : K[t] → K[t] as the linear map with

d/dt (tⁿ) = n · tⁿ⁻¹

(here n is an element of K, not just an integer — relevant in char p).

Theorem — Leibniz (Product) Rule
d/dt (fg) = f′g + fg′

Proof: Both sides are linear in f and g, so it suffices to check f = tⁿ, g = tᵐ. Then (fg)′ = (tⁿ⁺ᵐ)′ = (n+m)tⁿ⁺ᵐ⁻¹, and f′g+fg′ = ntⁿ⁻¹·tᵐ+tⁿ·mtᵐ⁻¹ = (n+m)tⁿ⁺ᵐ⁻¹. ∎

Theorem — Multiple Roots via GCD

gcd(f, f′) is non-constant ⟺ some irreducible q ∈ K[t] has q² | f.

Proof (⟸): If f = q²f₁, then f′ = 2qq′f₁+q²f₁′, so both f and f′ are divisible by q. ∎

Proof (⟹): Take irreducible q | gcd(f,f′). Write f=qh, so f′=q′h+qh′ — both divisible by q. So q | q′h. But deg q′ < deg q and q is irreducible, so q∤q′. Thus q | h, giving f = q²f₁. ∎

Application: Finding Multiple Roots
Find all λ ∈ ℂ such that f_λ(t) = t³−λt+1 has a multiple root

f_λ′(t) = 3t²−λ. A multiple root is a common root of f_λ and f_λ′. Division gives:

f_λ = (1/3)t · f_λ′ + [(−2/3)λt + 1]

The only candidate for a multiple root is t = 3/(2λ) (for λ≠0). Substituting into f_λ = 0:

f_λ(3/(2λ)) = 0 ⟺ 4λ³ = 27
λ ∈ { 3/∛4, (3/∛4)ζ, (3/∛4)ζ² } where ζ = e^{2πi/3} = −½ + (√3/2)i
Differentiation in Characteristic p
Note — Behaviour in Char p

If char K = p > 0 and f(x) = xᵖ−a, then f′(x) = p·xᵖ⁻¹ ≡ 0. So gcd(f,f′) is non-constant (= f) even though f may be irreducible! This corresponds to the inseparability phenomenon: f = (x−α)ᵖ has all roots equal.

Exam Preparation
Common Methods
Step-by-step procedures for the most frequently examined problem types — condensed from the full cheat sheet.
Method 1 — GCD in ℤ[i] with Bézout Coefficients
Setup: Find γ = gcd(α, β) and u, v ∈ ℤ[i] with γ = uα + vβ
  1. Run the division algorithm repeatedly. At each step write r₀ = q₁r₁ + r₂ where r₀=α, r₁=β. To divide α by β in ℤ[i]: compute α/β ∈ ℂ, round each of the real and imaginary parts to the nearest integer to get q, then r = α − qβ. Check N(r) < N(β).
  2. Continue until remainder = 0. The last non-zero remainder is γ (up to a unit factor). Normalise if needed (γ should have positive real part and non-negative imaginary part, or as instructed).
  3. Back-substitute (extended Euclid). Express each remainder rₖ = rₖ₋₂ − qₖrₖ₋₁, then substitute backwards to write γ = uα + vβ. Keep track of coefficients as 2×1 column vectors [s, t] where rₖ = s·α + t·β.
  4. List all GCDs. The GCDs of α and β are all unit multiples of γ: namely γ, iγ, −γ, −iγ (the four units of ℤ[i] are ±1, ±i).
Key formula: N(a+bi) = a²+b² Division: q = round(Re(α/β)) + i·round(Im(α/β)) Remainder: r = α − qβ, check N(r) < N(β)

Tip: Always verify your final GCD by checking N(γ) divides both N(α) and N(β), and verify uα+vβ = γ by direct computation.

Method 2 — Primary Decomposition & Smith Normal Form
Setup: Given A = Cₙ₁ × Cₙ₂ × … find primary decomp and SNF
  1. Factorise each order. For each nᵢ, write nᵢ = p₁^{a₁} · p₂^{a₂} · … (prime factorisation). Use CRT: Cₙ ≅ C_{p₁^{a₁}} × C_{p₂^{a₂}} × … when the prime-power factors are coprime.
  2. Group by prime. Collect all prime-power cyclic factors Cₚₖ together for each prime p. Sort each group's exponents in non-decreasing order.
  3. Primary decomposition is the full list of prime-power cyclic factors, e.g. C₄ × C₈ × C₃ × C₉ × C₇.
  4. Smith Normal Form (invariant factor form): Build the SNF diagonal entries d₁ | d₂ | … | dₖ by taking the largest prime-power from each prime group for d₁, the next largest for d₂, etc. (padding with 1s). Then dᵢ = product of chosen prime powers at position i.
  5. Check isomorphism: Two groups are isomorphic iff they have the same primary decomposition (equivalently, same SNF).
Example: C₁₂ = C₄ × C₃ (since gcd(4,3)=1) C₁₈ = C₂ × C₉ Primary: C₄ × C₂ × C₃ × C₉ SNF: d₁=lcm(4,2)·lcm(3)·… → d₁|d₂ rule: align prime powers, take column products

Divisibility check for SNF: Each dᵢ must divide dᵢ₊₁. For each prime p, list its exponents across all factors in non-decreasing order e₁≤e₂≤…; then p's contribution to dⱼ is p^{eⱼ}.

Method 3 — Chinese Remainder Theorem & Its Inverse
Setup: ℤ/(mn) ≅ ℤ/(m) ⊕ ℤ/(n) with gcd(m,n)=1 — find φ⁻¹(b,c)
  1. Find the Bézout identity. Use the Euclidean algorithm on m and n to find u, v ∈ ℤ with um + vn = 1.
  2. Write the inverse formula. The isomorphism φ: a+(mn) ↦ (a+(m), a+(n)) has inverse:
    φ⁻¹(b, c) = c·u·m + b·v·n (mod mn)
    Verify: c·um ≡ 0 mod m and c·um ≡ c mod n (since um ≡ 1 mod n because um+vn=1). Similarly b·vn ≡ b mod m.
  3. Substitute and reduce mod mn. Compute c·(um) + b·(vn), then reduce to the range {0, 1, …, mn−1}.
Key: um ≡ 1 (mod n) and vn ≡ 1 (mod m) φ⁻¹(b,c) = b·vn + c·um mod mn

Tip: Double-check by applying φ to your answer: it should map back to (b,c).

Method 4 — Zero Divisors and Units in ℤ/(n)
Setup: For a ∈ ℤ/(n), determine if a is a unit, zero divisor, or nilpotent
  1. Compute d = gcd(a, n).
  2. If d = 1: a is a unit. Find b = a⁻¹ mod n using the extended Euclidean algorithm: find u,v with ua + vn = 1, then b = u mod n.
  3. If d > 1 and d ≠ n: a is a zero divisor. Take b = n/d (reduced mod n). Then a·b = a·(n/d). Check: a·(n/d) = (a/d)·n ≡ 0 mod n since n | (a/d)·n. And b = n/d ≠ 0 mod n since d < n.
  4. If d = n (i.e. a ≡ 0): a is 0 itself — neither a unit nor a non-zero ZD.
a unit ⟺ gcd(a,n) = 1 a zero div ⟺ gcd(a,n) > 1 (and a ≢ 0) ZD witness: b = n/gcd(a,n) gives a·b ≡ 0 (mod n) Unit inverse: solve ua+vn=1 via Euclid, then b≡u (mod n)

Nilpotents in ℤ/(n): a is nilpotent iff every prime factor of n also divides a.

Method 5 — Detailed Division in ℤ[i]
How to divide α by β in ℤ[i] and find the remainder
  1. Compute α/β = α·β / N(β) ∈ ℂ. Recall: β = Re(β) − i·Im(β) and N(β) = Re(β)² + Im(β)².
  2. Write α/β = p/N(β) + i·q/N(β) for integers p, q. Round each rational part to the nearest integer: q₁ = round(p/N(β)), q₂ = round(q/N(β)). So q = q₁ + iq₂.
  3. Remainder: r = α − q·β. Verify N(r) < N(β).
α/β = α·conj(β) / N(β) q = round(Re(α/β)) + i·round(Im(α/β)) r = α − qβ, must have N(r) < N(β) Example: α=−9+19i, β=46+20i, N(β)=2116+400=2516 α·conj(β) = (−9+19i)(46−20i) = −414+180i+874i−380i² = (−414+380) + (180+874)i = −34+1054i...

Rounding tip: When the fraction is exactly x.5, round to the nearest even integer (or consistently to the same side — either convention gives a valid Gaussian integer quotient as long as N(r)<N(β)).
